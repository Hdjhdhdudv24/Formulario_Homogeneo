# Iconos PWA

Este directorio debe contener los iconos necesarios para la PWA:

- `icon-192.png` - Icono 192x192px
- `icon-512.png` - Icono 512x512px

## Generación de iconos

Puedes generar estos iconos desde el logo de Seguros Bolívar (`../assets/logo_SB.png`) usando herramientas como:

- [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- Herramientas de diseño como Photoshop, GIMP, etc.

Los iconos deben ser cuadrados (192x192 y 512x512) y preferiblemente con fondo transparente o con el color de marca.

## Nota temporal

Si los iconos no existen aún, la PWA seguirá funcionando pero no se mostrará el icono en la pantalla de inicio. Es recomendable generarlos antes del deploy a producción.
